Seguridad del Sistema

  • Administración de Roles
  • Administración de Usuarios

Administración de Roles: Este ítem permite la creación, edición, eliminación de los ROLES del sistema, cada ROL se le debe asignar sus propios privilegios, cada privilegio es permitir que el usuario pueda ingresar a un módulo o submodulo específico.


Administración de Usuarios: Este ítem permite la creación, edición, eliminación, restauración de contraseña de los USUARIOS del sistema, para la creación de los usuarios se debe tener definido el ROL que se les asignará.
