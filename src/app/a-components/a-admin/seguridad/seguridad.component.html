Seguridad del Sistema

  • Administración de Roles
  • Administración de Usuarios

Administración de Roles: Este ítem permite la creación, edición y eliminación para los ROLES del sistema, cada ROL se le debe asignar sus propios privilegios, cada privilegio es permitir que el usuario pueda ingresar a un módulo o submódulo específico.


Administración de Usuarios: Este ítem permite la creación, edición, eliminación y restauración de contraseña para los USUARIOS del sistema, para la creación del usuario se debe tener definido el ROL que se le asignará.
